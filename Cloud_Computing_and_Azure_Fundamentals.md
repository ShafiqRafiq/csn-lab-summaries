Lab: Cloud Computing and Azure Fundamentals

Objective:  
Gain an understanding of basic cloud computing concepts and how Microsoft Azure
provides cloud-based services.

Concepts Covered:  
- Cloud computing fundamentals  
- Public cloud vs on-premises environments  
- Virtual machines (VMs)  
- Resource groups  
- Cloud scalability and availability  

What I Worked On:  
- Explored the Azure portal and its main services  
- Learned how resources are organized using resource groups  
- Reviewed how virtual machines are created and managed in the cloud  
- Compared cloud-based systems to traditional on-premises setups  

What I Learned:  
- How cloud computing allows systems to scale on demand  
- Why cloud services reduce the need for physical hardware  
- How Azure organizes and manages cloud resources  
- The benefits of availability, flexibility, and cost efficiency in cloud environments
