# Linux

This folder contains Linux-related lab summaries and notes completed as part of
the Computer Systems Technician (CSN) program at Seneca Polytechnic.

The content focuses on foundational Linux system administration skills used in
entry-level IT and cybersecurity roles.

---

## Topics Covered
- Linux file system structure
- Command-line navigation and basic shell usage
- File and directory permissions
- User and group concepts
- Understanding root vs standard users
- Editing configuration files from the terminal

---

## Tools & Technologies
- Linux (CLI)
- Bash shell
- Standard Linux utilities

---

## Security & Real-World Relevance
Linux is widely used in servers, networking devices, and security tools.
Understanding permissions, users, and system structure is essential for
maintaining secure and stable systems.
